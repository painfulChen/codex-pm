# 隐私政策（草案）

生效日期：2026-02-13

本产品为本地优先的“Codex 项目管理工具”。默认情况下，数据仅保存在你的本机。

## 1. 我们收集什么

### 1.1 你在本机产生的数据（默认本地）
- 任务数据：任务ID、状态、优先级、项目归属、时间戳等（存于本机文件）。
- 终端心跳数据：agentId、标题、cwd、状态、最后在线时间等（存于本机文件）。
- 迭代日志：版本号、日期、标题、摘要、关联任务等（存于本机文件）。
- 可选 Git 快照：branch、commit、脏状态、变更文件列表（存于本机文件）。

### 1.2 匿名遥测（可选，默认关闭）
当你在“设置”里开启匿名遥测后，我们会在本机记录最小化的使用事件（例如：执行了哪些操作、耗时、错误码）。默认不记录任务正文、代码内容、完整文件内容。

你可以随时导出或清空本地遥测日志。

## 2. 我们如何使用数据
- 提供功能：看板、日志、项目文档同步、多终端治理。
- 改进体验（仅在你开启匿名遥测时）：用于统计高频功能与错误分布，帮助我们优化产品效率。

## 3. 数据如何共享
- 默认不向任何第三方共享或上传你的数据。
- 若未来你选择启用云同步/上传遥测，将以“显式开关 + 明示用途”的方式提供；你可以随时关闭。

## 4. 数据存储与保留
- 数据默认保存在本机目录下的 JSON/Markdown 文件中。
- 遥测日志默认保留在本机，可手动清空。

## 5. 你的权利
- 访问：你可以直接查看本机文件。
- 删除：你可以在产品内清空日志，或删除本机文件。
- 关闭：你可以随时关闭匿名遥测。

## 6. 联系方式
如有隐私问题，请联系产品维护者。
